position: static

The default position value. Elements flow naturally in the document. Top, right, bottom, left properties have no effect.

position: sticky

A hybrid of relative and fixed. Element is relative until it reaches a threshold, then becomes fixed.

Section 1

Scroll down to see the sticky header in action!

I'm Sticky! I'll stick to the top when you scroll.

Section 2

The sticky element behaves like relative positioning until you scroll past it.

Section 3

Then it becomes fixed to the specified position (top: 0 in this case).

Section 4

Keep scrolling to see it in action!

.sticky-element {
  position: sticky;
  top: 0;
  z-index: 10;
  /* Sticks when reaching top: 0 */
}
